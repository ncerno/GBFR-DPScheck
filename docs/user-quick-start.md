# 用户快速开始

更新时间：2026-06-04

正常用户只需要按应用首页的“首次使用”向导走。

## 第一次使用

1. 安装并启动 GBFR-DPScheck。
2. 如果没有 GBFR-ACT，点击“打开 GBFR-ACT 下载页”，下载后解压到固定目录。
3. 在向导里填写 GBFR-ACT 的 `act_ws.py` 路径；也可以直接填写 GBFR-ACT 文件夹。
4. 点击“保存并启动”。
5. 如果 Windows 弹出 UAC，允许 GBFR-ACT 管理员权限。
6. 点击“连接”。
7. 点击“打开 Overlay 小窗”。
8. 进入游戏打一轮木桩或任务，看到伤害后小窗会显示 DPS。

## 以后使用

1. 启动游戏。
2. 启动 GBFR-DPScheck。
3. 应用会按已保存路径尝试启动 GBFR-ACT 并自动连接 WebSocket。
4. Overlay 会自动打开；如果没有打开，点首页向导里的“打开小窗”。

## 需要手动处理的情况

- 没安装 GBFR-ACT：在向导里打开下载页，下载解压后填路径。
- 提示未找到 Python 3.11 64-bit：先安装 Python 3.11 64-bit，再回到向导点击“保存并启动”。
- WebSocket 未连接：确认游戏和 GBFR-ACT 已启动，再点“连接”。
- 提示路径不存在：重新填写 `act_ws.py` 或 GBFR-ACT 文件夹。
- Overlay 挡住鼠标：用主窗口顶部按钮开启或关闭鼠标穿透。
- 想看日志、Mock、Raw Events：进入设置页，展开“高级调试”。

## 不需要普通用户做的事

- 不需要安装 Node.js。
- 不需要安装 Rust。
- 不需要打开源码目录。
- 不需要手动编辑配置文件。
- 默认不需要保存 Raw Events。
