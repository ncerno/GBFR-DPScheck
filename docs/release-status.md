# 发布状态

更新时间：2026-06-04

## 当前结论

代码、文档和安装包构建链路已经完成。当前没有已知的代码阻塞项。

发布前还需要用户侧补充的只有实机验收记录，重点是干净 Windows 环境和长时间运行稳定性。

## 已完成

- 首页首次使用向导。
- GBFR-ACT 路径校验、后台启动和自动连接。
- 独立 Overlay 小窗、半透明背景、缩放、鼠标穿透和玩家名（角色名）。
- Dashboard、历史记录、配装测试和目标过滤。
- Raw Events 默认不落盘，改为手动采集。
- FAQ、隐私说明、第三方说明、普通用户快速开始和干净 Windows 验收清单。
- 团队增伤归因字段、组件和旧表述已移除。
- 安装包已重新生成。

## 待用户验收

- 干净 Windows 环境安装和启动。
- 缺少 WebView2 时的提示与恢复。
- 缺少 Python 3.11 64-bit 时的提示与恢复。
- GBFR-ACT 首次路径配置和后台启动。
- Overlay 在无边框、窗口化全屏、独占全屏下的可见性。
- 长时间真实战斗采集稳定性。
- 保存历史记录和配装测试后重启仍可读取。
- 卸载后用户数据目录是否保留，并在发布说明中告知。

## 可后续增强

- 历史记录批量删除。
- CSV 导出。
- 导入/导出文件选择器。
- 根据更多实战日志继续微调目标过滤规则。
- 如果 GBFR-ACT 后续提供地图名、BUFF/DEBUFF 归因等事件，再考虑扩展地图展示和团队增伤分析。

## 不纳入当前版本

- 不内置或再分发 GBFR-ACT。
- 不实现团队增伤归因。
- 不上传用户数据。
- 不读取或注入游戏进程。
